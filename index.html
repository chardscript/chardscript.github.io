ChardScript

Simple, Elegant, Easy Functional Programming.

Overview

The ChardScript programming language is easy to use and the Syntaxs are simple. Inspired by LuaScript and Syntax from BASIC and ShellScript. This is the place for you to learn how to use this programming language and learn. And here are all the features that ChardScript can do:

  • Run code quickly, no need to wait.
  • Support for PC operating systems.
  • Low capacity, stable computer.
  • Fully meet the needs of new developers.
  • Easy to use and easy to learn.
  • Free, no cost to buy.
