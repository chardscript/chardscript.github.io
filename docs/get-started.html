Welcome!

ChardScript is a powerful and fast programming language that is easy to learn and use and embed in your applications. ChardScript is designed to be a lightweight embeddable scripting language. See the introduction page for details and some reasons why you should choose ChardScript.

Get Started

Welcome! Are you completely new to programming? If not, we assume you're looking for information on why and how to get started with ChardScript. Fortunately, a programmer experienced in any programming language (whatever it may be) can pick up on ChardScript very quickly. It's also easy for beginners to use and learn, so use it!

Installation

Downloading ChardScript is very easy because they support all versions of Windows from 7 to 11, Linux, Mac or other operating systems. Download the latest version of ChardScript right here!
